# Coaching_Management
It's a coaching attendance application. Any teacher can use his coaching of coaching management system. He can check his all students records.

## Application Features
  - Admin can check attendence.
  - Admin can add user profile.
  - Admin can delete user profile.
  - Admin can update user profile.
  - Admin can upload classes file.
  - Admin can easily upload exam questions.
  - Admin can easily upload exam question solve.
  - Admin can easily upload his youtube videos.
  - Admin can easily upload his lecture sheets.
  - Admin can handle his class routine.
  - All features are free.

Any one can use this application.
